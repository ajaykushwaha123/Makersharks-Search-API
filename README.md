# Makersharks Search API

## Description
This project implements a search API for Makersharks, allowing buyers to search for manufacturers based on location, nature of business, and manufacturing processes.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL

## Prerequisites
- Java 22
- Maven
- MySQL
